# Stage 15 boundaries

Stage 15 implements scheduled tasks, recurring execution, private notifications, SMTP delivery, CalDAV event creation, inbound and outbound webhooks, external integration management, and automation history and control.

The stage does not implement a visual workflow builder, autonomous agents, unattended MCP tool execution, browser automation, inbound email polling, full calendar synchronization, OAuth provider catalogs, Redis, Celery, RabbitMQ, or multi-user automation ownership.
